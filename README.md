## 기능
- Work, Travel 카테고리 분리를 통해 할 일과 여행 목록 구분
- To do list 등록, 수정, 삭제 가능
- 실행한 list 체크 가능
- 앱 재실행 시 저장해둔 list 유지 및 마지막으로 본 카테고리 유지
